# Proof Packets

Proof packets are evidence folders for skill behavior. They should contain real
run artifacts: prompts, transcripts, rationalizations, patch notes, verifier
verdicts, and gate output. They are not marketing pages.

A complete packet usually contains:

```text
scenario.md
red-run.md
rationalizations.md
patch-summary.md
green-run.md
validator-verdict.md
gate-output.txt
final-summary.md
```

Use placeholders only when no public packet exists yet, and say that plainly.
Do not fabricate results.
